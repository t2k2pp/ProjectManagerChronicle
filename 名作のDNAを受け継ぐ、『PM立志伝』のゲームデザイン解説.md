# 名作のDNAを受け継ぐ、『PM立志伝』のゲームデザイン解説

### イントロダクション：面白いゲームは「面白い」の掛け算から生まれる

こんにちは。未来のゲームクリエイターである皆さんへ。多くの名作ゲームが、実は過去の優れたアイデアの「掛け算」から生まれていることをご存知ですか？一見すると全く異なるゲームの面白い要素を組み合わせることで、誰も見たことのない新しい体験が誕生するのです。

本稿では、カイロソフトの名作経営シミュレーション**『ゲーム発展国』**と、コーエーテクモゲームスの歴史シミュレーション**『太閤立志伝V』**という、二つの傑作のDNAを融合させて生まれた、新しい学習ゲームのコンセプトを解説します。その名も**『PM立志伝：プロジェクト・クロニクル』**。キャッチコピーは**「～デスマーチを回避し、伝説のPMを目指せ～」**です。

このゲームが目指すのは、単なる娯楽ではありません。プロジェクトマネジメント（PM）のリアルな知識と判断力を、最高のエンターテイメントを通じて**「楽しみながら、自然と学べる」**こと。この解説を通じて、そんな新しい学びの形と、ゲームデザインの面白さの一端を感じ取っていただければ幸いです。

--------------------------------------------------------------------------------

### 1. 設計の二本柱：『ゲーム発展国』と『太閤立志伝V』の「面白さの本質」

『PM立志伝』の設計思想を理解するために、まずはその源流となった二つの名作が持つ、全く異なる魅力の本質を解き明かしていきましょう。

#### 『ゲーム発展国』の面白さの本質

このゲームは、プレイヤーがゲーム会社の社長となり、組織を育てていくシミュレーションです。その魅力は、以下の3点に集約されます。

- **会社経営のサイクルが生み出す中毒性** 優秀な社員を集めてチームを作り、ゲームを開発し、世間から評価される。そして、その収益で会社をさらに大きくしていく…という、テンポの良い経営ループがもたらす高い中毒性が特徴です。
- **チーム育成の楽しさ** 個性豊かな社員たちを教育し、パラメータを上げ、時には「スーパーハッカー」のような上位職に転職させながら、自分だけの最強開発チームを作り上げる面白さがあります。
- **組み合わせを発見する喜び** 「RPG」と「ファンタジー」のように、相性の良いジャンルと内容の組み合わせを発見し、大ヒット作を生み出す発見の喜びも、このゲームの大きな魅力です。

#### 『太閤立志伝V』の面白さの本質

一方、このゲームは戦国時代を舞台に、一人の人物の人生を体験するシミュレーションです。その魅力は、組織運営とは異なる点にあります。

- **様々な人生を選べる圧倒的な自由度** 天下統一を目指す武士だけでなく、大商人、最強の剣豪、忍者、海賊など、様々な人生を自由に選んで体験できるロールプレイング性が最大の魅力です。
- **個人の成長を深く描く物語性** 主人公がスキルを磨き、各地で人脈を築き、一人の人間として立身出世していく物語性が描かれます。組織の一員としてだけでなく、個人としての成長が深く描かれます。
- **スキルカードを駆使する戦略的な交渉システム** 戦闘や外交交渉といった駆け引きを、スキルカードを駆使する戦略的なカードゲームで表現しており、知的な楽しさを提供します。

#### コンセプトの比較

二つのゲームのコンセプトを対比させると、その違いはより明確になります。

|   |   |   |
|---|---|---|
|比較項目|ゲーム発展国|太閤立志伝V|
|**主役**|会社（組織）|個人（主人公）|
|**目的**|最高のゲームを作り、会社を大きくする|自由な人生を送り、個人の目標を達成する|
|**コア体験**|経営・開発のループ|人生のロールプレイング|

『ゲーム発展国』が持つ**「組織運営の面白さ」**と、『太閤立志伝V』が持つ**「個人の成長物語の面白さ」**。この二つを掛け合わせることが、新しいゲームの骨格となるのです。

### 2. 構想の核：『組織運営』と『個人の成長』の融合

それでは、二つのゲームのDNAが、『PM立志伝』の心臓部（コアメカニクス）にどのように昇華されたかを解説します。

#### 『ゲーム発展国』のDNA：組織を動かす「運営のサイクル」

組織を俯瞰で見るマクロな視点は、『ゲーム発展国』の経営ループから着想を得ています。プロジェクトマネージャー（PM）がチーム全体を見渡し、リソースを管理する面白さを再現しました。

- **チーム編成とタスクのアサイン** 個性豊かなキャラクターたちをプロジェクトに配置し、誰にどのタスクを任せるかを考えるパズル的な楽しさを提供します。
- **進捗管理の可視化** ガントチャートや進捗バーが少しずつ伸びていく様子を見ることで、プロジェクトが進んでいる実感と満足感を得られます。
- **品質・コスト・納期（QCD）のバランス** 経営ゲームならではの「QCD」の概念を取り入れました。品質だけを追い求めればコストと納期が犠牲になり、納期だけを守ろうとすれば品質が犠牲になる。このトレードオフの判断が常に求められます。

#### 『太閤立志伝V』のDNA：個人を磨く「キャリアと交渉術」

PM自身の成長を追体験するミクロな視点は、『太閤立志伝V』の人生シミュレーションから着想を得ています。プレイヤーは単なる管理者ではなく、一人のプロフェッショナルとして成長していきます。

- **PMとしてのキャリアパス** 最初は下請けの小さな案件から始まり、経験を積んで大手へ転職したり、フリーランスとして独立したりと、一人のPMとしての人生を歩む物語性があります。
- **交渉カードバトル** 顧客やメンバーとの交渉は、スキルカードを使った戦略的なバトルで表現されます。
    - **『スコープ・クリープ』**: 相手の追加要求を「契約外」として防御する基本カード。
    - **『トレードオフ』**: 「納期を優先するなら品質を落とします」と提案し、交渉を有利に進めるカード。
    - **『エスカレーション』**: 上司を召喚し場の流れを変えるが、自身の評価も下がる諸刃の剣。
    - **『心理的安全性』**: 疲弊したチームの士気と生産性を回復させる強力なサポートカード。
- **スキルと資格の習得** プロジェクト経験を積むことでスキルツリーを解放したり、PMP®のような実在の資格試験に挑戦したりする、個人としての成長要素も用意されています。

このように、「組織運営（マクロ）」と「個人成長（ミクロ）」という二つの視点を組み合わせることで、単なる作業シミュレーターではない、物語性と戦略性に富んだ深みのあるゲーム体験が生まれるのです。

### 3. ゲームを彩る「人間」のリアリティが生む、奥深いゲーム性

プロジェクトマネジメントの本質は「機械の管理」ではなく「人の管理」にあります。このゲームで最も重要なのは、個性と人間味あふれるキャラクターたちの設計思想です。

#### ダブル・ヘキサゴン・システム

キャラクターの能力は、表と裏、二つの六角形（ヘキサゴン）で表現される独自のシステムを採用しています。

- **表の六角形（プロジェクトスキル）** プロジェクトの進捗や品質に直結する、業務遂行能力です。
    - **設計**: プロジェクト上流工程の要。この数値が高いと、「仕様漏れ」といった致命的な手戻りイベントの発生率が激減し、プロジェクトの安定性が向上します。逆に低いと、終盤にすべてが覆る悪夢を見るでしょう。
    - **製造**: 1ターンあたりに消化できる工数を決定する、チームの馬力そのもの。この数値が高いメンバーがいれば、ガントチャートのバーは目に見えて速く進みます。
    - **評価**: プロジェクトの最終品質を保証する「最後の砦」。この能力はバグ発見率に直結し、低いとリリース後に致命的な不具合が見つかり、会社の信頼は地に落ちます。
    - **折衝**: 顧客からの「仕様変更」や「納期短縮」といった攻撃からプロジェクトを守る防御力。技術力だけでは乗り切れない局面で真価を発揮します。
    - **提案**: 行き詰まった状況を打破する「ひらめき」の力。この数値が高いと、工数を大幅に削減するブレイクスルーイベントが発生し、絶望的な状況からの逆転劇を生み出します。
    - **判断**: 状況に応じてタスクの優先順位を正しく決定する自律性。この能力が高いメンバーは、PMが指示しなくてもクリティカルパスを優先するため、管理コストが激減します。
- **裏の六角形（チーム運営力）** プロジェクトの土台を支える、人間的な能力です。
    - **庶務**: 縁の下の力持ち。この能力を持つメンバーがいると、PMであるあなたの「行動ポイント」消費が軽減されます。彼らが経費精算などを片付けてくれるおかげで、あなたは本来のマネジメント業務に集中できるのです。
    - **幹事**: チームのムードメーカー。定期的に「飲み会」などの士気回復イベントを発生させ、デスマーチに陥ったチームの団結力を回復させる切り札となります。
    - **給仕**: メンバーの「スタミナ」の減少を緩和する癒やし手。残業が続いてもメンバーが体調不良で倒れるリスクを低下させ、プロジェクトの継戦能力を高めます。
    - **雑談**: メンバー間の対立を防ぐ潤滑油。スキルは高いが相性の悪いエース同士の間に配置することで、対立による進捗停止イベントを無効化できます。
    - **美貌**: その人間的魅力によって、周囲のパフォーマンスを底上げする特殊能力。自分より美貌値が高い相手の前で「良いところを見せたい」という心理を誘発し、能力にボーナスを与えます。
    - **幸運**: 論理では説明できない奇跡を引き起こす力。普段は何の効果もありませんが、ごく低確率で「クライアントの都合で無理な要求が撤回される」といったラッキーイベントを発生させます。

この二層構造の意図は、**「一見役に立たないような人が、実はチームの危機を救う」**という、現実の職場における「あるある」を再現するためです。

#### 二つ名を持つNPCと「やりくり上手」の腕の見せ所

ゲームには、特別な能力を持つ固有NPCが登場します。彼らは当初、**『深夜のコミット怪人』**のように会議に全く出ない問題児や、**『微笑みの仕様粉砕者』**のようになんでも笑顔で受け入れてくれる理想的な顧客に見えるかもしれません。しかし、PMとして彼らの本質を理解し、信頼を得ることで真の能力が解放され、プロジェクトの最強の味方となるのです。

ですが、覚えておいてください。このゲームに完璧なチームは決して作れません。能力が高いキャラクターは、人月単価（コスト）が法外に高かったり、**『ドキュメント嫌い』**や**『独断専行』**といった強烈な欠点を持っていたりする**「諸刃の剣」**です。限られた予算、納期、そして凸凹な個性を持つメンバーという制約の中で、いかに成果を出すか。この**「やりくり」**こそが、PMとしての腕の見せ所であり、このゲームの面白さの核心なのです。

このゲームは単なる能力値の足し算ではありません。個性豊かな人間たちが引き起こす予測不能な「化学反応」をマネジメントする、**人間臭いシミュレーションゲーム**なのです。

### 4. 結論：最高の学びは、最高のエンターテイメントから

ここまで解説してきた通り、『PM立志伝：プロジェクト・クロニクル』は、『ゲーム発展国』が持つ**「中毒性のある経営ループ」**と、『太閤立志伝V』が持つ**「自由な人生設計と個人の成長」**という偉大なDNAを受け継いでいます。そして、そこにPMという仕事の面白さ、大変さ、そして何よりも「人間臭さ」を凝縮しました。

このゲームでは運の要素は最小限に設計されており、成功の鍵は**「正しい知識と判断」**です。プレイを通じて、あなたは**WBS**や**クリティカルパス**、**ステークホルダーマネジメント**、**EVM**といった、プロジェクトマネジメントの世界標準（PMBOK）に基づいた思考を、単なる用語としてではなく、勝利のための戦略として自然と身につけるでしょう。

**最高の学びは、最高のエンターテイメントから生まれる。**

このゲームが、未来の優れたゲームクリエイターやプロジェクトマネージャーを一人でも多く育てる一助となることを願っています。さあ、あなたのプロジェクト・クロニクル（年代記）を始めましょう。
