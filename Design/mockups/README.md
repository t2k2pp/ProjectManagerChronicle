# UIモックアップ一覧

各画面のHTMLプロトタイプは `Design/mockups/` ディレクトリに格納されています。

## 実装済みモックアップ

### 1. [index.html](./index.html)
モックアップ一覧のインデックスページ。すべてのモックアップへナビゲート可能。

### 2. [dashboard.html](./dashboard.html)
**社長室（ダッシュボード）画面**
- 現在のプロジェクト状況表示
- クイックアクションボタン
- 組織状態のサマリ
- 通知・案件パイプライン
- **技術**: ガラスモーフィズム、グリッドレイアウト、ホバーアニメーション

### 3. [pm_cockpit.html](./pm_cockpit.html)
**PMコックピット（メイン画面）**
- オフィスビュー（キャラクター管理）
- ガントチャート（WBS進捗）
- ドラッグ&ドロップによるアサイン機能
- EVM指標表示（SPI/CPI）
- 週間方針選択・ターン進行ボタン
- **技術**: Drag and Drop API、SVG、動的スタイル変更

### 4. [character_detail.html](./character_detail.html)
**キャラクター詳細画面**
- ダブル・ヘキサゴンチャート（SVGで実装）
- 基本情報・現在の状態
- 特性タグ表示
- 信頼度・二つ名システム
- **技術**: SVGレーダーチャート、モーダルオーバーレイ、アニメーション

### 5. [card_battle.html](./card_battle.html)
**カ��ドバトル（交渉）画面**
- 対戦アリーナ（プレイヤーvs相手）
- ゲージシステム（納得・不満）
- 5種類のカードタイプ（防御、交渉、攻撃、支援、特殊）
- 対話ログ表示
- カードホバーエフェクト
- **技術**: グラデーション、パルスアニメーション、カスタムカードデザイン

## モックアップの使用方法

1. `index.html` をブラウザで開いてください
2. 各モックアップカードをクリックすることで、対応する画面プレビューにアクセスできます
3. すべてのモックアップはスタンドアロンで動作し、JavaScriptによる簡易的なインタラクションを実装しています

## 技術仕様

- **フレームワーク**: Pure HTML/CSS/JavaScript（実装時はReact移植予定）
- **スタイル手法**: 
  - ガラスモーフィズム（半透明 + backdrop-filter: blur）
  - グラデーション背景
  - カスタムアニメーション
- **レスポンシブ**: デスクトップ最適化（1920x1080推奨）
- **ブラウザ互換性**: Chrome 90+, Edge 90+, Firefox 88+
