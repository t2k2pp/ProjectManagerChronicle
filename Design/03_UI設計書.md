## 1. デザインコンセプト

### 1.1 ビジュアルテーマ

**「PMコックピット」- プロフェッショナルな管理画面**

- **カラーパレット**: ビジネスライクかつモダン
  - Primary: ディープブルー (#1e3a8a)
  - Secondary: アンバー (#f59e0b)
  - Success: グリーン (#10b981)
  - Warning: イエロー (#eab308)
  - Danger: レッド (#ef4444)
  - Background: ダークグレー (#1f2937) / ライトグレー (#f3f4f6)

- **タイポグラフィ**: 
  - 見出し: 'Noto Sans JP', Bold
  - 本文: 'Noto Sans JP', Regular
  - 数値: 'Roboto Mono', Monospace

- **UI スタイル**: 
  - ガラスモーフィズム（半透明とブラー効果）
  - ネオモーフィズム（立体的な影）
  - グリッドベースレイアウト

### 1.2 デザイン原則

1. **情報密度の最適化**: 重要な情報を一目で把握できる
2. **階層構造の明確化**: 見出し、本文、補足の視覚的差別化
3. **アクセシビリティ**: WCAG 2.1 AA準拠（コントラスト比4.5:1以上）
4. **一貫性**: すべての画面で統一されたデザイン言語
5. **フィードバック**: ユーザーアクションに対する即座の視覚的応答

#### カード
```typescript
// Standard Card
className="bg-white dark:bg-gray-800 rounded-xl shadow-lg p-6 border border-gray-200 dark:border-gray-700"

// Glass Card (ガラスモーフィズム)
className="bg-white/10 backdrop-blur-lg rounded-xl border border-white/20 shadow-xl p-6"
```

#### バッジ
```typescript
// Status Badge
className="px-3 py-1 rounded-full text-xs font-bold uppercase"
// Success: bg-green-100 text-green-800
// Warning: bg-yellow-100 text-yellow-800
// Danger: bg-red-100 text-red-800
```

- **リスク**: ⚠️

```
|                            |                                                |
|  【オフィスビュー】         |  【ガントチャート】                            |
|  +--------+  +--------+    |  要件定義 ████████░░ 80%  👤佐藤              |
|  | 👤佐藤 |  | 👤田中 |    |  設計     ██████░░░░ 60%  👤田中              |

|  | HP:60  |  | HP:100 |    |                                                |
|  | 😰     |  | 😊     |    |                                                |

|                            |                                                |
```

**インタラクション**:
- キャラカードクリック → 詳細モーダル
- ガントチャートのタスクバーへキャラをドラッグ&ドロップ
- 「1週間進める」ボタン → ターン処理開始（プログレスバー表示）

**状態表示**:
- HP（体力）ゲージ: 緑(80-100) / 黄(50-79) / 赤(0-49)
- 表情アイコン: 😊(高士気) / 😐(普通) / 😰(低士気) / 💤(休暇中)
- リスクアイコン: 💀(危険タスク、判断スキル保有者のみ表示)

## 5. インタラクション設計

### 5.1 マウスインタラクション

| 要素 | ホバー | クリック | ドラッグ |
|------|--------|----------|----------|
| ボタン | スケール1.05、影拡大 | リップルエフェクト | - |
| キャラカード | 枠光る | 詳細モーダル表示 | ガントチャートへアサイン |
| タスクバー | 情報ツールチップ | 選択状態 | 順序変更 |
| カード | 拡大 | 選択/使用 | - |

### 5.2 キーボードショートカット

- `Space`: ターン進行
- `Esc`: モーダル閉じる
- `Ctrl+S`: セーブ
- `1-5`: キャラクター1～5の詳細表示
- `Tab`: 次のタスクへフォーカス移動

### 5.3 フィードバック

#### 成功時
- 緑のチェックマーク ✅
- ポジティブなサウンド
- 「+10」などの数値がフロート表示

#### 失敗時
- 赤のバツマーク ❌
- 画面振動エフェクト
- 注意音

#### 警告時
- 黄色の警告マーク ⚠️
- パルス（点滅）アニメーション
