# 名作のDNAから生まれる『プロジェクトマネージャー体感ゲーム』の作り方

### 導入：面白さの掛け算から生まれる、新しい学びの形

多くの名作ゲームは、過去の優れたアイデアの「掛け算」から生まれています。一見すると全く異なるゲームの面白い要素を組み合わせることで、誰も見たことのない新しい体験が誕生するのです。

本稿では、カイロソフトの名作経営シミュレーション**『ゲーム発展国』**と、コーエーテクモゲームスの歴史シミュレーション**『太閤立志伝V』**という、二つの傑作のDNAを融合させ、新しい「プロジェクトマネージャー（PM）体感学習ゲーム」を構想するプロセスを解説します。

このゲームが目指すのは、単なる娯楽ではありません。プロジェクトマネジメントのリアルな知識と判断力を、最高のエンターテイメントを通じて**「楽しみながら、自然と学べる」**こと。そんな新しい学びの形を、一緒に作り上げていきましょう。

--------------------------------------------------------------------------------

### 1. 設計の二本柱：『ゲーム発展国』と『太閤立志伝V』とは？

まずは、今回の構想の元になった二つのゲームを知らない方のために、それぞれの「面白さの本質」を分かりやすく解説します。この二つのゲームが持つ、全く異なる魅力の源泉を理解することが、新しいゲームデザインの第一歩です。

- **『ゲーム発展国』の面白さの本質:**
    - **会社経営のサイクル:** 優秀な社員を集めてチームを作り、ゲームを開発し、世間から評価される。そして、その収益で会社をさらに大きくしていく…という、テンポの良い経営ループがもたらす高い中毒性。
    - **チーム育成の楽しさ:** 個性豊かな社員たちを教育し、パラメータを上げ、時には「スーパーハッカー」のような上位職に転職させながら、自分だけの最強開発チームを作り上げる面白さ。
    - **組み合わせの妙:** 「RPG」と「ファンタジー」のように、相性の良いジャンルと内容の組み合わせを発見し、大ヒット作を生み出す発見の喜び。
- **『太閤立志伝V』の面白さの本質:**
    - **圧倒的な自由度:** 天下統一を目指す武士だけでなく、大商人、最強の剣豪、忍者、海賊など、戦国時代を舞台に様々な人生を自由に選んで体験できるロールプレイング性。
    - **個人の成長体験:** 主人公がスキルを磨き、各地で人脈を築き、一人の人間として立身出世していく物語性。組織の一員としてだけでなく、個人としての成長が深く描かれる点。
    - **カードバトルによる交渉:** 戦闘や外交交渉といった駆け引きを、スキルカードを駆使する戦略的なカードゲームで表現し、知的な楽しさを提供するシステム。

この二つのゲームのコンセプトを対比させると、その違いはより明確になります。

|   |   |   |
|---|---|---|
|特徴|ゲーム発展国|太閤立志伝V|
|**主役**|会社（組織）|個人（主人公）|
|**目的**|最高のゲームを作り、会社を大きくする|自由な人生を送り、個人の目標を達成する|
|**コア体験**|経営・開発のループ|人生のロールプレイング|

『ゲーム発展国』が持つ**「組織運営の面白さ」**と、『太閤立志伝V』が持つ**「個人の成長物語の面白さ」**。この二つを掛け合わせることで、私たちが目指す新しいゲームの骨格が見えてきます。

### 2. 構想の核：経営シムと人生シムの融合

それでは、二つのゲームシステムが、新しいPM体感ゲームのコアメカニクス（ゲームの心臓部）にどのように昇華されたかを解説します。

#### 『ゲーム発展国』のDNA：組織を動かす「運営のサイクル」

組織を俯瞰で見るマクロな視点は、『ゲーム発展国』の経営ループから着想を得ています。

- **チーム編成とタスクのアサイン** 個性豊かな社員たちをプロジェクトに配置し、誰にどのタスクを任せるかを考えるパズル的な楽しさ。
- **進捗管理の可視化** ガントチャートや進捗バーが少しずつ伸びていく様子を見ることで、プロジェクトが進んでいる実感と満足感を得られます。
- **品質・コスト・納期のバランス** 経営ゲームならではの「QCD（Quality, Cost, Delivery）」の概念。品質だけを追い求めればコストと納期が犠牲になり、納期だけを守ろうとすれば品質が犠牲になる…このトレードオフの判断が常に求められます。

#### 『太閤立志伝V』のDNA：個人を磨く「キャリアと交渉術」

PM自身の成長を追体験するミクロな視点は、『太閤立志伝V』の人生シミュレーションから着想を得ています。

- **PMとしてのキャリアパス** 最初は下請けの小さな案件から始まり、経験を積んで大手SIerへ転職したり、フリーランスとして独立したりと、一人のPMとしての人生を歩む物語性。
- **交渉カードバトル** 『太閤立志伝V』のカードシステムを応用し、PM業務の駆け引きを再現します。顧客やメンバーとの交渉は、スキルカードを使った戦略的なバトルで表現されます。例えば、追加要求を防御する**『スコープ・クリープ』**や、納期と品質の交換を提案する**『トレードオフ』**といった基本カードに加え、上司を召喚して場の流れを変える**『エスカレーション』**（ただし自身の評価も下がる諸刃の剣）、疲弊したチームの士気を回復させる**『心理的安全性』**のような強力なカードを駆使して、困難な状況を乗り切ります。
- **スキルと資格の習得** プロジェクト経験を積むことでスキルツリーを解放したり、PMP®のような実在の資格試験に挑戦したりする、個人としての成長要素。

このように、**組織を運営するマクロな視点**と、**PM個人として成長するミクロな視点**を組み合わせることで、単なる作業シミュレーターではない、物語性と戦略性に富んだ深みのあるゲーム体験が生まれるのです。

### 3. ゲームを彩る「人間」のリアリティ

プロジェクトマネジメントの本質は「機械の管理」ではなく「人の管理」にあります。このゲームで最も重要なのは、個性と人間味あふれるキャラクターたちの設計思想です。

#### ダブル・ヘキサゴン・システム

キャラクターの能力は、表と裏、二つの六角形（ヘキサゴン）で表現されます。

- **表の六角形（プロジェクトスキル）**
    - `設計`: 手戻りを防ぐ、緻密な上流工程を構築する力。
    - `製造`: プロジェクトを着実に推進する、高速な実装力。
    - `評価`: バグを発見し、製品の品質を担保する力。
    - `折衝`: 顧客の要求からプロジェクトを守る、粘り強い交渉力。
    - `提案`: 困難な状況を打破する、革新的なアイデアを生み出す力。
    - `判断`: 状況に応じて正解を選び取る、的確な意思決定力。
- **裏の六角形（ヒューマン力）**
    - `庶務`: 経費精算などをこなし、PMの管理コストを削減する力。
    - `幹事`: チームの士気を回復させ、組織の団結力を高める力。
    - `給仕`: メンバーの体力を維持し、心身の健康を守る力。
    - `雑談`: メンバー間の対立を防ぐ、潤滑油としての対話力。
    - `美貌`: 周囲を奮起させ、チームの能力を底上げする人間的魅力。
    - `幸運`: 理屈では説明できない、奇跡的な成功を引き寄せる力。

この二層構造の意図は、**「一見役に立たないような人が、実はチームの危機を救う」**という、現実の職場における「あるある」を再現するためです。

#### 二つ名を持つ固有NPC

『太閤立志伝V』の有名武将にヒントを得て、ゲーム内には特別な能力を持つ固有NPCが登場します。彼らは最初、扱いにくい問題児に見えるかもしれません。しかし、PMとして彼らの本質を理解し、信頼関係を築くことで、**『深夜のコミット怪人』**や**『遺産を守る者』**といった「二つ名」が解放され、真の能力を発揮します。これは、人材マネジメントの奥深さを学ぶ最高の教材となります。

#### 「やりくり上手」こそ腕の見せ所

能力が高いキャラクターは、人月単価（コスト）が高かったり、『ドキュメント嫌い』のような強烈な欠点を持っていたりします。完璧なチームは決して作れません。限られた予算、納期、そして凸凹な個性を持つメンバーという制約の中で、いかに成果を出すか。この**「やりくり」**こそが、PMとしての腕の見せ所であり、このゲームのパズル的な面白さの核心です。

結論として、このゲームは単なる能力値の足し算ではありません。個性豊かな人間たちが引き起こす予測不能な「化学反応」をマネジメントする、人間臭いシミュレーションゲームなのです。

### 4. PMの日常を再現する「現実」のスパイス

ゲームのリアリティと学習効果をさらに高めるため、現実のPM業務に根差した「日常のスパイス」を加えています。

- **12ヶ月カレンダーと季節イベント** 「ゴールデンウィークで休みたいメンバー」と「遅れたスケジュール」。『太閤立志伝V』が季節の変化で戦略性を生んだように、このゲームでは日本の年度カレンダーを導入。「夏季休暇」や「年末年始」といった、リソース調整に頭を悩ませるPMの胃が痛くなるタスクを、戦略的なイベントとしてゲーム化しました。
- **主人公（プレイヤー）のメイキング** ゲーム開始時に「20代の若手」を選ぶか、「50代のベテラン」を選ぶかで、プレイスタイルは全く異なります。若手は体力で無理が効きますが、スキルや判断力は乏しい。ベテランは体力がない代わりに、豊富な経験とスキルで部下を的確に動かします。プレイヤー自身のキャリアプランを追体験できるのです。
- **週次（1ターン＝1週間）のゲームサイクル** PMの仕事は、毎日細かい指示を出すことではありません。このゲームでは1ターンを1週間とし、**「週初めに計画を立て、週末に結果を振り返り、次週の計画に活かす」**という、本来のマネジメントのリズムを再現。日々の作業に追われるのではなく、大局観を持ってプロジェクトを導く体験を提供します。

### 5. 結論：最高の学びは、最高のエンターテイメントから

ここまで解説してきた通り、この『プロジェクトマネージャー体感ゲーム』は、『ゲーム発展国』が持つ**「中毒性のある経営ループ」**と、『太閤立志伝V』が持つ**「自由な人生設計と個人の成長」**という偉大なDNAを受け継いでいます。そして、そこにPMという仕事の面白さ、大変さ、そして何よりも「人間臭さ」を凝縮しました。

このゲームでは、運の要素は最小限です。成功への道は、常に**「正しい知識と判断」**によって切り拓かれます。プレイを通じて、WBS、クリティカルパス、ステークホルダーマネジメントといったPMBOKに基づいた思考が自然と身につくでしょう。

最高の学びは、最高のエンターテイメントから生まれる。 この構想が、未来の優れたプロジェクトマネージャーを一人でも多く育てる一助となれば、開発者としてこれ以上の喜びはありません。
